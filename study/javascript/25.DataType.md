# 자바스크립트 - 데이터 타입

## 원시 데이터 타입과 객체
- 원시 데이터 타입 : 기본 데이터 타입(숫자, 문자, 불리언, null, undefined)
- 객체 데이터 타입 : 참조 데이터 타입(원시 데이터 타입 외 전부)

## 래퍼 객체
> 원시 데이터 타입을 감싸는 객체
원시 데이터 타입은 객체가 아니기 때문에 해당 변수에 속성이나 메소드를 추가할 수 없다. 하지만 래퍼 객체를 사용하면 속성이나 메소드를 추가 할 수 있다.
- 숫자 : Number
- 문자열 : String
- 불리언 : Boolean
- null : X
- undefined : X
